# AutoScrollViewPager
自动滚动的无限轮播ViewPager

## 效果
![截图](https://raw.githubusercontent.com/peerless2012/AutoScrollViewPager/master/ScreenShots/ScreenGif.gif)

## 已经实现的功能
1. 无限轮播。
2. 手触摸以后跳过轮播（最好是忽略手触摸的时间）。
3. 忽略手动滑动切换上一个或下一个。
